= YDCMD =
The actual source code and documentation can be found at:
https://github.com/joshua1955/ydcmd

For Windows you may need to update the list of trusted certification
authorities (ca-root.crt), which can be found at:
http://curl.haxx.se/docs/caextract.html

Before you begin it is recommended to create a configuration file:
C:\Users\<Your User Name>\.ydcmd.cfg

On some systems, the program may require to install Microsoft Visual C++
Redistributable Package which can be found at:
https://support.microsoft.com/en-us/help/2977003/the-latest-supported-visual-c-downloads
